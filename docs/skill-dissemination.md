# Skill Dissemination

How to share and distribute Agent Skills so others can discover and use them.

## Directories & Marketplaces

Submit your skill to these platforms for visibility:

| Platform | URL | Notes |
|----------|-----|-------|
| **Skills Directory** | [skillsdirectory.com](https://skillsdirectory.com) | 3,500+ indexed skills, [submit here](https://skillsdirectory.com/submit) |
| **Agent Skills Repo** | [agentskillsrepo.com](https://agentskillsrepo.com) | Curated, categorized |
| **MCP Market** | [mcpmarket.com/tools/skills](https://mcpmarket.com/tools/skills) | Skills + MCP servers |

## Checklist

Before sharing your skill:

- [ ] Valid `SKILL.md` with clear `name` and `description`
- [ ] Passes validation: `npx skills-ref validate ./your-skill`
- [ ] README with installation instructions
- [ ] License file (MIT, Apache-2.0, etc.)
- [ ] GitHub topics added
- [ ] Submitted to at least one directory
